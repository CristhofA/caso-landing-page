Opening-tools

El Inspector de Elementos está disponible en la mayoría de los navegadores modernos. Aunque su acceso varía ligeramente entre ellos, en todos los casos es muy sencillo abrirlo. Aquí te mostramos cómo hacerlo en los navegadores más populares:

Google Chrome​

  • Haz clic derecho sobre cualquier elemento de la página y selecciona "Inspeccionar".
  • También puedes usar el atajo de teclado:
    • Ctrl + Shift + I (Windows/Linux)
    • Cmd + Option + I (Mac)
  • Otra forma: haz clic en el menú (tres puntos verticales) → Más herramientas → Herramientas para desarrolladores.

Mozilla Firefox​

  • Haz clic derecho y selecciona "Inspeccionar".
  • O presiona:
    • Ctrl + Shift + I (Windows/Linux)
    • Cmd + Option + I (Mac)
  • También puedes ir al menú (≡) → Más herramientas → Desarrollador web → Inspector.

Microsoft Edge​

  • Igual que en Chrome:
    • Clic derecho → "Inspeccionar"
    • O usa Ctrl + Shift + I o Cmd + Option + I.
  • Menú → Más herramientas → Herramientas para desarrolladores.

Safari (macOS)​

🧠 Nota: Primero debes habilitar el menú de desarrollo.

Para habilitarlo:​

  1. Ve a Safari > Preferencias > Avanzado.
  2. Marca la casilla "Mostrar el menú de desarrollo en la barra de menús".

Luego:​

  • Haz clic derecho → "Inspeccionar elemento"
  • O presiona Cmd + Option + I.

Consejo útil​

🔁 Puedes mantener el Inspector abierto mientras navegas por distintas páginas. Esto es útil si quieres analizar cómo cambia el contenido dinámicamente o revisar múltiples vistas.

Con estas combinaciones, abrir el Inspector será cuestión de segundos, sin importar el navegador que uses.
