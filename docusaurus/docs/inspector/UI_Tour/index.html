UI Tour

La interfaz del Inspector de Elementos puede parecer compleja al principio, pero está diseñada para ayudarte a entender y modificar fácilmente el contenido de una página web. A continuación, te mostramos un recorrido general por sus principales componentes:

🧭 Barra superior​

Aquí encontrarás pestañas que te permiten cambiar entre diferentes herramientas. Algunas de las más comunes son:

  • Elements (Elementos): Para inspeccionar y editar HTML/CSS.
  • Console (Consola): Para ver errores, ejecutar JavaScript y mensajes de depuración.
  • Sources (Fuentes): Para ver y depurar el código fuente JavaScript.
  • Network (Red): Muestra todas las solicitudes de red (imágenes, APIs, archivos, etc.).
  • Performance: Para analizar el rendimiento de la página.
  • Application, Security, Lighthouse, etc.: Otras herramientas avanzadas según el navegador.

🧱 Panel de elementos (HTML + CSS)​

Cuando abres la pestaña Elements, verás dos secciones principales:

  • Estructura HTML (izquierda): Muestra el DOM (Document Object Model) de la página. Puedes hacer clic en los elementos para inspeccionarlos.
  • Estilos CSS (derecha): Muestra los estilos aplicados al elemento seleccionado. Puedes editar estos estilos en tiempo real.

📜 Consola​

La pestaña Console permite escribir y ejecutar comandos en JavaScript. También muestra errores, advertencias y mensajes de console.log().

📶 Panel de red​

En la pestaña Network puedes ver:

  • Todas las solicitudes HTTP realizadas por la página.
  • Tiempos de carga.
  • Estado de las respuestas (200, 404, etc.).
  • Archivos como imágenes, scripts, estilos, etc.

📊 Otras herramientas útiles​

  • Performance: Graba cómo se comporta la página mientras la usas para detectar cuellos de botella.
  • Application: Inspecciona el almacenamiento (cookies, localStorage), service workers y caché.
  • Security: Revisa si la conexión es segura (HTTPS, certificados).
  • Lighthouse: Genera reportes de accesibilidad, performance, SEO y más.

🧩 Personalización​

Puedes mover las pestañas, cambiar su posición (vertical/horizontal), o incluso desacoplar el panel del navegador.

Esta interfaz está diseñada para ser flexible y poderosa, permitiéndote modificar, probar y analizar cualquier aspecto de un sitio web en tiempo real.
